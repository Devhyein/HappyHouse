<h1>Happy House</h1>

<h3>기획 배경</h3>
	본 프로젝트는 이사를 가기 위해 인터넷으로 집을 알아보는 사람들을 타깃으로 하는 집 소개 페이지를 제작하는 	것을 목표로 하였다.
	사용자가 원하는 집의 위치 뿐만 아니라, 주변 환경, 다른 분양정보들을 참고할 수 있게 하여 편리하게 집을 찾을 	수 있도록 도와주는 페이지를 제작하고 싶어 프로젝트를 기획하게 되었다.
	

<h3>목표</h3>
 - 공공데이터를 정제하여 가져올 수 있다..
 - 원하는 정보를 크롤링하여 가져올 수 있다.
 - 가져온 데이터들을 XML을 이용해 파싱할 수 있다.
 - 외부 API를 사용할 수 있다. 
 - 차트로 데이터를 나타낼 수 있다.
 - 부트스트랩을 이해하고 적용할 수 있다.


<h3>개발 환경</h3>
 - 개발 언어 : JAVA, JSTL, HTML, JavaScript, XML, JQuery, JSP
 - 개발 툴 : STS, VScode
 - 개발 환경 : [OS] Windows10, [Browser] Chrome, [WAS]
